MessagingToolとは

    MessagingToolは、PCやワークステーション間でメッセージを交換するためのツールです。メッセージを交換する為のサーバーを必要とせず、MessagingToolを動作させているPCやワークステーションに直接メッセージを送ることができます。主な機能は、
    • 複数の相手へ同時にメッセージを送信することができます。
    • 接続されているサブ・ネットワークへメッセージの同報を行うことができます。
    • 席を離れて留守にする場合には、「留守録メッセージ」を残すことができます。他の人はあなたが残した留守録メッセージを取り出す事が可能です。
    • MessagingToolを動作させている同僚の状態を容易に知ることができ、また、留守であるかも容易に知ることができます。
    • 仮想的な電子会議室を作ることができます。
    • ファイルを直接転送することができます。
    • Swing版では、デスクトップにスティキー・ノート（付箋紙)を貼ることが出来ます。
ダウンロード
    MessagingToolを構成するファイルは、msgtool.jarファイルにすべてまとめてあります。Windows 95/98/NTの場合には、自己解凍形式になっていますので、ダウンロードに解凍してmsgtool.jarファイルを取り出し下さい。
     
    • msgtool.jar
    • Windows 95/98/NT 4.0(自己解凍exeファイル)

起動方法
  • JDK1.1.8用AWTバージョン

    • MessagingToolを起動するにはJRE 1.1.8(Java Runtime Environment)が必要ですので、JavaSoftからダウロードしてインストールして下さい。その際に、Windows95/NT環境の場合にはjre1_1_8-win-i.exeをダウンロードして下さい。JREがインストールされましたら、以下の方法で起動できます。
    　
      Unix:
        % cd <msgtool.jarをインストールしたディレクトリ>
        % jre -cp msgtool.jar MessagingTool &
        　
      Windows 95 / 98 / NT:
        > cd <msgtool.jarをインストールしたディレクトリ>
        > jrew -cp msgtool.jar MessagingTool
        　
    (注) jreおよびjrewコマンドがサーチパス上に無い場合には、フルパスで指定して下さい。
     
  • JDK1.1.8用Swingバージョン

    • Swing版を使用するには、JRE1.1.8とSwing1.1.1が必要です。Swing版を動作させるには、Swing 1.1.1に含まれるswingall.jarファイルが必要です。msgtool.jarとswingall.jarファイルを一つのディレクトリーに入れたら、以下の方法で起動で出来ます。
     
      Unix:
        % cd <msgtool.jarとswingall.jarをインストールしたディレクトリ>
        % jre -cp msgtool.jar:swingall.jar MessagingTool -swing &
        　
      Windows 95 / 98 / NT:
        > cd <msgtool.jarとswingall.jarをインストールしたディレクトリ>
        > jrew -cp msgtool.jar;swingall.jar MessagingTool -swing
    (注) jreおよびjrewコマンドがサーチパス上に無い場合には、フルパスで指定して下さい。
     
  • JDK1.2.2用AWTバージョン

    • MessagingToolを起動するにはJRE 1.2.2(Java Runtime Environment)が必要ですので、JavaSoftからダウロードしてインストールして下さい。その際に、Windows95/NT環境の場合には国際版であるjre1_2_2-win-i.exeをダウンロードして下さい。JREがインストールされましたら、以下の方法で起動できます。
    　
      Unix:

        % java -cp msgtool.jar MessagingTool &
        　
      Windows 95 / 98 / NT:

        > java -cp msgtool.jar MessagingTool
        　
    (注) javaコマンドがサーチパス上に無い場合には、フルパスで指定して下さい。
     
  • JDK1.2.2用Swingバージョン

    • MessagingToolを起動するにはJRE 1.2.2(Java Runtime Environment)が必要ですので、JavaSoftからダウロードしてインストールして下さい。その際に、Windows95/NT環境の場合には国際版であるjre1_2_2-win-i.exeをダウンロードして下さい。JREがインストールされましたら、以下の方法で起動できます。

使用方法
    日本語版のユーザーガイド(MS Word形式)は、まだドラフト版で完成していません。申し訳ありませんが、英語版も一緒に参照願います。なお、ダウロードされたmsgtool.jarファイルには、英語版と日本語版の両方のMessagingToolが含まれていますので、日本語環境で動作させれば、メニューやメッセージ等はすべて日本語で表示されます。
