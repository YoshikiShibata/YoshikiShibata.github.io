『新世代Javaプログラミングガイド』正誤表

2020年4月13日改訂

はじめに

『新世代Javaプログラミングガイド版』は、何度も校正を重ねましたが、誤りを見逃していたりするかもしれません。誤りに気づかれた方は、連絡をお願いいたします。

第1刷 (2020年3月11日発行)
  • iii頁：下から3行目の「Class Data Sharing」（コードフォント）を「Class Data Sharing」（イタリック）に修正（2020.4.13）
  • 4頁：コード例にある「LocalDateTime dateTime = new LocalDateTime.now()」（二か所）を「LocalDateTime dateTime = LocalDateTime.now()」に修正（2020.3.20）
  • 56頁：コード例にある「age > 10 ? "Kid" : "Not a Kid"; 」（二か所）を「age > 10 ? "Not a Kid" : "Kid";」に修正（2020.3.20）
  • 56頁：コード例にあるコメント「// StringとList<person>が入力、Optionalが戻り値」（二か所）を「// StringとList<Person>が入力、Optionalが戻り値」に修正（2020.3.20）
  • 65頁：下から7行目の「・すべてのオブジェクトはヒープに保存されますオブジェクトを参照するローカル変数 ...」を「
    ・すべてのオブジェクトはヒープに保存されます。
    ・オ    ブジェクトを参照するローカル変数 ...
    」に修正（2020.3.20）
  • 75頁：コード例にあるコメント「// HttpClientはHttpRequestsを送信して、HttpResponseを利用可能にします」を「// HttpClientはHttpRequestを送信して、HttpResponseを利用可能にします」に修正（2020.3.20）
  • 76頁：8行目と10行目の「getHttpClient」を「newHttpClient」に修正（2020.3.20）
  • 80頁：三つ目のコード例中の「.timeout(Duration.ofSeconds(240)」を「.timeout(Duration.ofSeconds(240))」に修正（2020.3.20）
  • 82頁：上から5行目（コード例は除く）の「BodyHandlers.discard()を使って」を「BodyHandlers.discarding()を使って」に修正（2020.3.20）
  • 82頁：上から14行目（コード例は除く）の「discard」を「discarding」に修正（2020.3.20）
  • 87頁：一番下のコード例中の「main(String[] uri)」を「main(String[] args)」に修正（2020.3.20）
  • 87頁：一番下のコード例中の「info@ejavaguru@gmail.com」を「info.ejavaguru@gmail.com」に修正（2020.3.20）
  • 128頁：コード例中の「case BOTTLE: damage = 20;」を「case BOTTLE: damage += 20;」に修正（2020.3.20）
  • 158頁：コード例中のコンストラクタ宣言「public Emp(String name, int age)」を「public EmpPOJO(String name, int age)」に修正（2020.3.20）
  • 158頁：コード例中のメソッド宣言「public void setName()」を「public void setName(String name)」に修正（2020.3.20）
  • 158頁：コード例中のメソッド宣言「public void setAge()」を「public void setAge(int age)」に修正（2020.3.20）
  • 163頁：上から10行目（コード例を除く）の「デコパイル」を「デコンパイル」に修正（2020.3.20）
  • 181頁：最初のコード例中の「key -> key.uppercase();」を「key -> key.toUpperCase();」に修正（2020.3.20）
  • 182頁：下から4行目（コード例は除く）の「toUppercaseメソッド」を「toUpperCaseメソッド」に修正（2020.3.20）
